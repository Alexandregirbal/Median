---------------- REQUETES -----------------

¤ MARKS:

	GET | les notes d'un USER

		SELECT DISTINCT us.EmailUser, IdMark, MarkM, ScaleM, CoefM, ma.IdSubject, ue.SemesterUE, ue.IdUE 
		FROM marks ma, ues ue, users us, subjects su
		WHERE ma.EmailUser=us.EmailUser AND su.IdSubject=ma.IdSubject AND ue.IdUE=su.IdUE
		AND us.EmailUser='---USER---'
		ORDER BY su.IdSubject 

	POST | une nouvelle mark avec les paramètres compréhesibles suivants
		INSERT INTO `marks` (`IdMark`, `MarkM`, `ScaleM`, `CoefM`, `IdSubject`, `EmailUser`) 
		VALUES (NULL, '16', '20', '3', '1', 'hugo.brando@etu.umontpellier.fr');
	
	DELETE | /!\ si vide /!\
		DELETE FROM `marks` 
		WHERE `marks`.`IdMark` = ---VAR---

	PUT | une note identifiée par IDMARK en changeant differents parametres VAR au choix avec les diffrents éléments

		UPDATE `marks` SET `MarkM` = '---VAR---' WHERE `marks`.`IdMark` = ---IDMARK---
